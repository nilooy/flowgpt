## Built With

- Next.js
- langchain

- DaisyUI

## Installation

1. Clone the Repo

2. `cd` into the folder
```sh
cd flowgpt
```

3. Install all deps with yarn
```sh
yarn
```

4. Run in development mode
```sh
yarn dev
```
